# Movie Ticket Booking System

Welcome to the Movie Ticket Booking System repository! This project provides a simple command-line interface for booking and managing movie tickets.

## Overview

The Movie Ticket Booking System allows users to:

- View and edit the price of movie tickets (admin only).
- View reserved tickets (admin only).
- Purchase tickets for different movies.
- Cancel reserved tickets.
- Add and remove movies (admin only).

## Features

- Price of tickets can be adjusted by the admin using a password.
- Users can purchase tickets for available movies.
- Admin can view details of reserved tickets.
- Users can cancel their reservations.
- Admin can add and remove movies from the system.

## Usage

1. Choose options from the main menu to interact with the system.
2. For admin features, a password is required.
